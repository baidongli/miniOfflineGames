# iOS Nearby Connections Bridge

Swift framework wrapping Google's `NearbyConnections` CocoaPod, exposing a
flat C ABI to Unity via `[DllImport("__Internal")]`.

## Build

The framework is built via Xcode workspace generated by CocoaPods:

```
pod install
xcodebuild -workspace NearbyBridge.xcworkspace -scheme NearbyBridge \
  -configuration Release -sdk iphoneos
```

Copy the produced `.framework` (and required `.xcframework` dependencies)
into `unity/Assets/Plugins/iOS/`.

## API surface

C functions are declared with `@_cdecl` in Swift, matching the signatures in
`unity/Assets/Networking/Transport/NearbyConnectionsTransport.cs`. Callbacks
into Unity use `UnityFramework`'s `UnitySendMessage` (or function-pointer
callbacks set via init).

## Info.plist additions (in Unity build settings)

- `NSLocalNetworkUsageDescription`
- `NSBonjourServices` (entries for the Nearby service)
- `NSBluetoothAlwaysUsageDescription`
